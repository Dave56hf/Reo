# Reo Wallet

Landing page for Reo Wallet, built with React and Vite.

## Tech Stack
- React 19
- Vite 7
- ESLint 9
- react-icons

## Scripts
- `npm run dev`: Start local dev server
- `npm run build`: Build production bundle
- `npm run preview`: Preview production build
- `npm run lint`: Run lint checks

## Notes
- The page is split into reusable section components (`Header`, `Showcase`, `About`, `HowIt`, `Testimonials`, `Footer`).
- Navigation anchors map to section IDs for smooth in-page navigation.
- Global styles and focus-visible defaults live in `src/index.css`.
